# 📊 Workforce Planning & Productivity Dashboard (Excel)

## 🚀 Project Overview
This project analyzes **workforce productivity, attendance, and labor utilization** using Microsoft Excel.  

It demonstrates how data-driven insights can support:
- Smarter workforce planning  
- Better staffing decisions  
- Improved employee performance optimization  

---

## 📈 Key KPIs
- 👥 Total Employees  
- 📅 Attendance Rate  
- ⚡ Productivity Score  
- ⏱️ Overtime %  

---

## ✨ Features
- 📊 Interactive dashboard using Pivot Tables & Charts  
- 🧩 Workforce segmentation by:
  - Department  
  - Training Levels  
  - Overtime  
  
- 📌 KPI-based performance analysis  
- 🎛️ Interactive filtering using Slicers  
- 🧮 Custom calculated metrics:
  - Productivity  
  - Efficiency

## 🔍 Key Insights
- High overtime may indicate **staffing inefficiencies**  
- Productivity varies significantly across departments  
- Training hours have a direct impact on performance  
- Attendance plays a critical role in overall output
