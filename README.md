Workforce Planning & Productivity Dashboard (Excel)
This project analyzes workforce productivity, attendance, and labor utilization using Excel. It demonstrates how data can support workforce planning, staffing decisions, and performance optimization.

Key KPIS List:

Total Employees
Attendance Rate
Productivity Score
Overtime %

Features
Pivot Tables and Charts
Workforce segmentation (Department, Training, Overtime)
Performance analysis
Interactive filtering (Slicers)
Calculated KPIs (Productivity, Efficiency)

Key Insights 
High overtime indicates potential staffing inefficiencies
Productivity varies across departments
Training hours impact performance
Attendance directly affects output
